# 竞彩赛事赔率解析器

## 功能说明

本技能用于从 okooo.com 竞彩网站提取足球赛事数据，包括：

- 比赛编号（如 005）
- 联赛名称（如友谊赛、J 联赛等）
- 比赛日期和时间
- 对阵双方（主队 VS 客队）
- 胜平负赔率
- 让球赔率

## 输出格式

按日期分组展示赛事信息：

```
2026-03-28 星期六
005|友谊赛|22:00|奥地利 VS 加纳（赔率：胜 1.45 平 3.90 负 5.45）| 奥地利 -1 VS 加纳（赔率：胜 2.48 平 3.30 负 2.37）
006|友谊赛|22:00|南非 VS 巴拿（赔率：胜 1.93 平 2.93 负 3.70）| 南非 -1 VS 巴拿（赔率：胜 4.30 平 3.40 负 1.66）

2026-03-29 星期日
001|J 联赛|13:00|町田 VS 川崎（赔率：胜 1.79 平 3.56 负 3.45）| 町田 -1 VS 川崎（赔率：胜 3.42 平 3.60 负 1.79）
```

## 技术实现

- **编码处理**: 使用 gb2312 解码中文网页内容
- **数据解析**: 正则表达式提取 HTML 中的比赛数据
- **日期分组**: 按比赛日期分组并显示星期几
- **赔率格式化**: 清晰展示胜平负和让球赔率

## 使用方法

```bash
python3 main.py
```

## 注意事项

1. 需要网络连接访问 okooo.com 网站
2. 网站编码为 gb2312，需正确解码
3. 赔率数据实时变化，以网站为准
4. 仅展示有完整赔率的比赛
